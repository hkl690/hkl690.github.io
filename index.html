Predicting Future Contaminant Levels with Machine Learning

This project aims to predict future contaminant levels in drinking water using machine learning. Data from the EPA Unregulated Contaminant Monitoring Rule (UCMR) is used to train machine learning models to forecast future contaminant levels based on historical data.
